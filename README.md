# 🏥 Détection et Classification du Cancer de la Peau par Deep Learning

## 📌 Présentation
Application intelligente basée sur le Deep Learning permettant la détection et la classification automatique des lésions cutanées à partir d’images dermatologiques.

---

## 🎯 Objectif
- Détecter les lésions cutanées
- Classifier en 9 catégories
- Aider au diagnostic précoce
- Intégrer IA + application web

---

## 🧠 Dataset
- ISIC Archive
- Images dermatologiques annotées
- 9 classes de maladies
- Prétraitement + augmentation des données

---

## 🏷️ Classes
- Actinic Keratosis
- Basal Cell Carcinoma
- Dermatofibroma
- Melanoma
- Nevus
- Pigmented Benign Keratosis
- Seborrheic Keratosis
- Squamous Cell Carcinoma
- Vascular Lesion

---

## ⚙️ Technologies

### IA
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Scikit-learn
- OpenCV

### Web
- React.js
- Node.js
- Express.js

---

## 🏗️ Architecture
Frontend (React) → Backend (Node.js / Express) → Modèle IA (TensorFlow) → Résultat
