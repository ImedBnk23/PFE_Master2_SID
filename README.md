## 📌 Présentation

Ce projet propose une solution basée sur le **Deep Learning** pour la détection et la classification automatique des lésions cutanées à partir d’images dermatologiques.

L’objectif est d’aider au diagnostic précoce du cancer de la peau à travers une application web intelligente.

---

## 🎯 Objectifs

- Détection automatique des lésions cutanées  
- Classification en 9 types de maladies  
- Amélioration du diagnostic précoce  
- Intégration IA dans une application web  
- Développement d’un système complet (Full Stack)

---

## 🧠 Dataset

- Source : ISIC Archive  
- Images dermatologiques annotées  
- 9 classes de lésions cutanées  
- Prétraitement + augmentation des données  

---

## 🏷️ Classes

- Actinic Keratosis  
- Basal Cell Carcinoma  
- Dermatofibroma  
- Melanoma  
- Nevus  
- Pigmented Benign Keratosis  
- Seborrheic Keratosis  
- Squamous Cell Carcinoma  
- Vascular Lesion

## 🚀 Installation

### 1️⃣ Cloner le projet
```bash
git clone https://github.com/ton-compte/PFE_Master2_SID.git
cd PFE_Master2_SID
```

### 2️⃣ Backend
```bash
cd app_web/backend
npm install
npm start
```

### 3️⃣ Frontend
```bash
cd app_web/frontend
npm install
npm run dev
```
